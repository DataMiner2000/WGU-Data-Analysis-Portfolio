This portfolio highlights the work Caleb Camilleri completed as part of the required Udacity Data Analyst Nanodegree for WGU's Data Analytics program. The projects included here represent my hands-on experience applying data analysis, statistical methods, and machine learning techniques to real-world datasets. Together, they showcase my development in data cleaning, exploratory analysis, modeling, visualization, and building reproducible analytical workflows using industry-standard tools.

## Projects:

## Deploying a Scalable ML Pipeline with FastAPI

Tools: Python, scikit-learn, FastAPI, Pydantic, pytest 

Description: Finished building and then deployed an income-prediction ML model behind a FastAPI service with testing, model persistence, and slice-based performance reporting.

## Build an ML Pipeline for Short-Term Rental Prices

Tools: Python, MLflow, Weights & Biases, Hydra, Random Forest 

Description: Developed a reusable end-to-end ML pipeline for recurring rental-price model retraining with modular ETL and experiment tracking.

## Finding Donors for CharityML (Supervised Learning)

Tools: Python, pandas, NumPy, scikit-learn 

Description: Trained and evaluated supervised learning models to predict high-income individuals for donor targeting using feature engineering and model selection.

## Identify Customer Segments (Unsupervised Learning)

Tools: Python, pandas, NumPy, scikit-learn (PCA, clustering), seaborn, Matplotlib 

Description: Applied unsupervised learning to segment customers and identify overrepresented demographic groups for targeted marketing strategy.

## Real-World Data Wrangling with Python

Tools: Python, pandas, KaggleHub/API data retrieval 

Description: Gathered, cleaned, and merged multi-source traffic accident data to resolve data quality issues and answer a time-based risk analysis question.

## Analyze A/B Test Results

Tools: Python, pandas, NumPy, Matplotlib, statistical testing 

Description: Evaluated treatment vs. control conversion performance to support data-driven product decision-making from A/B experiment outcomes.

## Investigate a Dataset: Medical No-Show Appointments

Tools: Python, pandas, NumPy, Matplotlib 

Description: Conducted EDA on 100K+ medical appointment records to examine no-show patterns, including SMS reminder and health-related effects.

## Explore Bike Share Data (R)

Tools: R, ggplot2 

Description: Compared bikeshare trip-duration patterns across cities, subscriber types, and gender using integrated datasets and exploratory visualization.
